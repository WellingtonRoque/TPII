Este repositório contém os materiais, exemplos e evolução do projeto desenvolvido durante a disciplina **Técnicas de Programação II**.

A disciplina segue um modelo **Bootcamp**, onde os alunos desenvolvem progressivamente um **sistema web completo**, aplicando conceitos de:

* Programação Orientada a Objetos
* Arquitetura de Software
* Padrões de Projeto
* Persistência de Dados
* Desenvolvimento Web

---

# 🎯 Objetivo da Disciplina

Capacitar os alunos a desenvolver aplicações web organizadas e escaláveis utilizando **Python, Flask e SQLAlchemy**, aplicando boas práticas de engenharia de software.

Ao final do curso, o aluno terá desenvolvido um **sistema web funcional**, incluindo:

✔ Arquitetura MVC
✔ Banco de dados com ORM
✔ CRUD completo
✔ Templates dinâmicos
✔ Layout profissional
✔ Padrões de projeto
✔ Testes básicos
✔ Versionamento com Git

# 🛒 Projeto do Curso

Durante o semestre será desenvolvido um sistema chamado:

## **Sistema Web de Catálogo de Produtos**

O sistema permitirá:

* cadastro de produtos
* edição de produtos
* exclusão de produtos
* busca e filtros
* organização profissional do código
* persistência em banco de dados

O sistema será evoluído ao longo das aulas.

# ▶️ Como executar o projeto

1️⃣ Clone o repositório

```
git clone https://github.com/seu-repositorio
```

2️⃣ Entre na pasta do projeto

```
cd projeto
```

3️⃣ Instale as dependências

```
pip install flask flask-sqlalchemy
```

4️⃣ Execute a aplicação

```
python app.py
```

5️⃣ Abra no navegador

```
http://localhost:5000
```
